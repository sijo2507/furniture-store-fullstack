# Santhosh Foam & Furnishings

Full-Stack E-Commerce Web Application built using React, Node.js, Express, and MongoDB.

## Features
- Product listing and categorization
- Dynamic product details with size variations
- Backend API integration
- Scalable architecture

## Tech Stack
Frontend: React.js  
Backend: Node.js, Express  
Database: MongoDB  

## Folder Structure
frontend/ – React application  
backend/ – Node + Express API
